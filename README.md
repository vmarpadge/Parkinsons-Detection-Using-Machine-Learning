# Parkinson's Disease Detection Using Machine Learning

Detects Parkinson's disease from voice recordings using an SVM (Support Vector Machine) classifier. The system analyzes phonation features — the sounds produced when pronouncing sustained vowels — since phonation is the most affected component of speech in Parkinson's patients.

## How It Works

1. **Feature Extraction**: Voice recordings (`.wav`) are processed using [PRAAT](https://www.fon.hum.uva.nl/praat/) (via `parselmouth`) to extract acoustic features: jitter, shimmer, NHR, HNR, pitch statistics, pulse counts, and period measurements.
2. **Training**: An SVM classifier (RBF kernel, C=10) is trained on the [UCI voice dataset](https://archive.ics.uci.edu/ml/datasets/Parkinson+Speech+Dataset+with++Multiple+Types+of+Sound+Recordings) (1040 recordings from 56 subjects). Feature scaling is applied via `StandardScaler`, and the scaler is saved alongside the model.
3. **Prediction**: New voice samples are analyzed, features are extracted and scaled using the saved scaler, then classified as Parkinson's-positive or healthy.

## Setup

```bash
# Python 3.8+ required
pip install -r requirements.txt
```

## Usage

### Train the model

```bash
python train.py
```

Outputs `svmclassifier.pkl` containing both the trained SVM model and the `StandardScaler`.

### Predict from a .wav file (CLI)

```bash
python predict.py path/to/voice_recording.wav
```

### Predict using the GUI

```bash
python gui.py
```

Use **File → Open** to select a `.wav` file, then click **Detect**.

## Dataset

- **Source**: UCI Machine Learning Repository
- **Samples**: 1040 recordings from 56 subjects (sustained vowels 'a' and 'o', 3 times each)
- **Features**: 24 acoustic features including jitter variants, shimmer variants, autocorrelation, noise-to-harmonics ratio, pitch statistics, pulse/period measurements
- **Label**: Binary (1 = Parkinson's, 0 = Healthy) at column index 28

## Selected Features (10 of 24)

The classifier uses these 10 features (by column index): Jitter(ddp), SD of period, Mean period, NTH, Jitter(local), Shimmer(local dB), Jitter(local absolute), Jitter(ppq5), Shimmer(apq3), Jitter(rap).

## Notes

- The prediction scripts require `parselmouth` which provides a Python interface to PRAAT for audio analysis.
- Voice recordings should contain sustained vowel phonation ('a' or 'o') for best results.
- The model achieves ~65% accuracy on `final2.csv` (the original `final1.csv` dataset, which likely achieved ~85-90%, is no longer available). 5-fold cross-validation is used during training.
